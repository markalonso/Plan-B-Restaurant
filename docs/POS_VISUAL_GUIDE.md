# POS System - Visual Guide

## Access
The POS system is protected by authentication. Users must sign in through the admin login at `/admin/login`.

![Admin Login](https://github.com/user-attachments/assets/c7b5405d-db99-4f0b-aed8-039b6ef25a1c)

## Navigation
After login, access the POS system from the admin sidebar:
- Look for the highlighted "POS" button (green background)
- Available from any admin page
